# Premier Site PHP

Ce projet est un site web simple créé pour être une démonstration de base de la création d'un site avec PHP. Il sert d'introduction aux concepts de PHP et au développement web dynamique, en incluant des fonctionnalités comme un formulaire de contact et un traitement basique des données.

## Fonctionnalités

- **Page d'accueil dynamique** : Un message personnalisé est affiché en fonction des interactions avec l'utilisateur.
- **Formulaire de contact** : Permet aux utilisateurs de soumettre des informations via un formulaire simple.
- **Traitement des données** : Le projet inclut des exemples de traitement de données envoyées via formulaire.
